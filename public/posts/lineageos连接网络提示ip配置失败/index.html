IP配置失败

连接所有网络都提示IP配置失败，通过配置静态IP也连不上网络，感觉就是WIFI模块不能用了。

使用 Magisk root 后就这样了，LineageOS 17.1（wayne），重新刷回刷机包的 boot.img 后就没问题了。

但是如果需要用到 root 权限怎么办？

以 Root 身份进入 ADB Shell

首先，进入设置中【开发者选项】，开启【Root 身份的调试】

接着执行下面的命令获取 root 权限

# 启用 adb root 身份
adb root
restarting adbd as root

# 以 root 身份进入
adb shell

# 验证 Root 权限
whoami
root  # 返回结果为 root，说明你已经成功以 root 身份进入 ADB shell。
